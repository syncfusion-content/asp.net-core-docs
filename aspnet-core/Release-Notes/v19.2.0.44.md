## Common

### Breaking Changes
{:#common-breaking-changes}

* The Syncfusion `ReportViewer`, `ReportWriter`, and `ReportDesigner` component packages, scripts, and samples are no longer shipped with the Essential Studio setup in the `v19.2.0.44` release. It is available as part of the <a href="https://help.boldreports.com/embedded-reporting/aspnet-core-reporting/">Bold Reports</a> product and is not included in the Essential Studio.
