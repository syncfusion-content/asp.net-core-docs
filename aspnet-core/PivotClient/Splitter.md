# Splitter

You can resize Cube Dimension Browser and Axis Element Builder by setting `enable-splitter` property to true.This property is disabled by default.

{% highlight CSHTML %}

<ej-pivot-client id="PivotClient1" enable-splitter="true">
   //..
</ej-pivot-client>

{% endhighlight %}
