## Methods








### destroy()


destroy the linear gauge all events bound using this._on will be unbind automatically and bring the control to pre-init state.

{% highlight html %}

<ej-linear-gauge id="LinearGauge">
</ej-linear-gauge>

var gauge = $("#LinearGauge").data("ejLinearGauge");
gauge.destroy();

{% endhighlight %}
