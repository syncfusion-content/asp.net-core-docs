# Overview

The ASP.NET Core Gantt control is designed to visualize and edit the project schedule, and track the project progress. It helps to organize and schedule the projects and also you can update the project schedule through interactions like editing, dragging and resizing.

**Key Features**

* Sorting
* Editing
* Task relationship
* Stripline
* Baseline
* Timeline customization
* Work break down structure
* Taskbar customization
* Task label customization
* Localization
* Resources
